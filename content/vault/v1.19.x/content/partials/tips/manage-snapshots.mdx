<Tip title="Backup and restore data">

All Vault editions support snapshot save and restore features for data backup
and restoration. Vault Enterprise users benefit from automated snapshots to
local or cloud storage, and individual secret recovery.

Reference the following resources for more details on snapshots.

- [Manage snapshots](/vault/docs/sysadmin/snapshots) guide.
- [`snapshot` CLI command](/vault/docs/commands/operator/raft#snapshot)
- [`/sys/storage/raft/snapshot-load` endpoint](/vault/api-docs/system/storage/raftsnapshotload#load-a-snapshot-into-vault)

</Tip>
