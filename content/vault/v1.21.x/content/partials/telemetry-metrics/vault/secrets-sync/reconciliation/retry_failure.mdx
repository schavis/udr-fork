### vault.secrets-sync.reconciliation.retry.failure ((#vault-secrets-sync-reconciliation-retry-failure))

Metric type | Value  | Description                                                              
----------- | ------ | ------------------------------------------------------------------------
counter     | number | Total number of failed reconciliation retry attempts by failure category

Retry failure metrics include the following labels:

- `destination_type` - the type of sync destination.
- `cause` - the category of failure: `internal` for Vault-side errors, or `external` for client-side errors.
