### vault.secrets-sync.operation ((#vault-secrets-sync-operation))

Metric type | Value    | Description
----------- | -------- | -----------
counter     | number   | Tracks the number of sync and unsync operations performed on secrets to external destinations. This metric increments each time a secret is synced to or unsynced from a destination.

The metric includes the following labels:

- `operation` - The type of operation performed, for example, `sync` or `unsync`.
- `status` - The status of the operation, for example, `synced`, `unsynced`, `internal-error`, `external-service-error`, or `client-side-error`.
- `destination-type` - The type of destination store, for example, `aws-sm`, `azure-kv`, `gcp-sm`, `gh`, or `vercel-project`.
